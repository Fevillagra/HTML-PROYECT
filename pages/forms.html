forms <form method=""> </form>

Etiqueta de bloque; sobre la que se coloca controles interactivos normalmente hay etiquetas de input contenidas en ella.
El metodo especifica como sera enviada la informacion
Method get Por dejecto, get” indica que los datos enviados se adjuntarán en la barra de direcciones del cliente, al final de la url correspondiente y después de un signo de interrogación de cierre. Si se envía más de un dato, éstos irán separados por el símbolo &. El método GET solicita una representación de un recurso específico. Las peticiones que usan el método GET sólo deben recuperar datos.
Methodpost es aquel que envia al servidor (metodo seguro) El método POST se utiliza para enviar una entidad a un recurso en específico, causando a menudo un cambio en el estado o efectos secundarios en el servidor.

input <input type="tipo" name="" value="">

label <label for="mismo id que input o name">

Etiqueta de linea; mediante la cual se le solicita informacion al usuario, recibe un atributo nombre para que pueda interactuar con el servidor.
Atributo name igual que id se usa para ser referenciado en Js Atributo placeholder es lo que aparecera en un color tenue en el lugar que se escribe
Dentro de type(tipos de input) puede haber:

  1. text Espera entrada de texto, por defecto aparece type="text"
  2. password Espera entrada de texto no visible para teclear contraseñas y claves
  3. number Contiene dos botones para elegir numeros
  4. email Espera la entrada de un texto con formato email
  5. color Se puede seleccionar un color
  6. range min="numero" max="numero" Nos permite elegir un numero dentro de un rango mediante una interfaz donde el seleccionador esta sobre un slide
  7. date Sirve para que el usuario seleccione una fecha
  8. time Sirve para que el usuario seleccione una hora
  9. button value="boton" Es un boton, las palabras contenidas en el estaran especificadas en el atributo value
  10. submit Es un boton para enviar el servidor la informacion
  11. Required="" Es un atributo que hace requisito que el input sea completado en caso de que el usuario desee enviarlo
  12. <textarea>texto<textarea> Es una etiqueta de linea; para escribir texto en ella.

  13. El atributo que aloja el texto escrito en el imput es value
  14. action="" Es el atributo con el url a donde se mandara la pagina(form)
  15. radio Un circulo que se puede seleccionar
  16. checkboxUna caja que se puede seleccionar
  17. autofucus="" (atributo)Pone el cursor de manera automatica en el especificado
  18. <datalist id="colors">
    <option value="text">
    </datalist> especifica una lita de opciones a elegir a un input text el id de datalist va igual que el valor del atributo list="colors"

Progreso <progress min="0" max="100" value="50"> </progress>

Etiqueta de línea; márca el progreso en un proceso

Atributo min Marca desde donde empieza a contar el progreso.

Atributo max Marca el total necesario para completar el progreso.

Atributo value Marca el progreso actual
