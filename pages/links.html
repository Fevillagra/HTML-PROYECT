Si es externa el link recibira http o htpps(hipertext transfer protocol)


Links relativos

Si deseara entrar a una carpeta que esta en mi mismo nivel sería: href="carpeta/index.tml"

Si está una carpeta atrás sería href="../index.html"

Si está en la misma carpeta sería href="./index.html o index.html"

Multimedia

Imagen <img src="RUTA" alt="Nombre alternativo">

Etiqueta de línea; el atributo src recibe la ruta, y el nombre alternativo Aparece solo en caso de que sea un navegador para no videntes o que el formato no sea soportado. Si decido usar el atributo width no debo darle atributo height ya que la imagen se ajusta completamente a una de las dos, si especifico las dos se deformara

El atributo title="info" Sirve para que cuando se pase el cursor sobre la imagen aparecera un titulo de la imagen

Video <video src="Ruta" controls=""> </video>

Etiqueta de bloque, el atributo controls le da controles al video. Entre sus Etiquetas debería haber un mensaje que aparecera en caso de que el navegador no tolere el formato

Se puede emplear la etiqueta source(etiqueta abierta) para determinar la ruta

Atributo: autoplay El video comienza a reproducirse a penas se cargue.

Atributo: loop El video se reproduce de manera indefinida

Audio <audio src="Ruta" controls=""> </audio>

Etiqueta de bloque; la etiquetas <source src="audio.mp3" type="audio/mpeg/"> y <source src="audio.ogg" type="audio/ogg"> sirven para especificar la ruta.Entre sus Etiquetas debería haber un mensaje que aparecera en caso de que el navegador no tolere el formato.

atributo autoplay="" Cuando se define este atributo, el audio comienza a reproducirse tan pronto como esté listo, sin pedir permiso al visitante.

el atributo loop="" Sirve para reproducir de manera indefinida
