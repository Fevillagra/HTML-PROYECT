¿Qué es html?

Según sus siglas "Lenguaje de Marcas de hipertexto"

Cuando se crea una página la primera debe recibir el nombre de index.html ya que el servidor buscara este archivo en primer lugar.

Elementos de html

Etiquetas: Van entre simbolos mayor y menor <Etiqueta> CONTENIDO </Etiqueta>

Atributos: Van dentro de las llaves y tienen asignado un valor <Etiqueta atributo="valor">. Cuando el atributo no requiere un valor no es necesario usar los simbolos ="" por lo tanto la disposicion sería la siguiente <Etiqueta atributo>

Tipos de Etiquetas:

Abiertas: Solo requieren una etiqueta, no dos. Ej: <hr>

Cerradas: Requieren de ambas etiquetas <Etiqueta> </Etiqueta>

Estructura basica

<!DOCTYPE html> Sirve para especificar al navegador la version de html usada, en este caso se trata de la ultima version.

<html> Etiqueta principal y es donde va a estar contenida toda la página, tanto lo que podemos ver como lo que no podemos ver de la página

    <head> Etiqueta que aloja todo aquello que no se ve en el cuerpo principal de la página

    </head>

    <body> Todo aquello que es "visible" de la página

    </body>

</html>

Nota: Los archivos html pueden terminar con html o htm
